This plugin allows you to design and customize the woocommerce checkout. 

== Description ==

The plugin takes the standard checkout design one step further.

The plugin allows you to customize your checkout layout with 6 amazing templates.

The Main Features:

Free Version:
1.4 Build-in templates.
2.Inputs fields Customization: reorder ,width of the inputs,font colors and background color.
3.Positioning of the “checkout form” and the “order review” position.

Full Version:
1.6 Build-in templates.
2.Inputs fields Customization: reorder ,width of the inputs,font colors and background color.
3.Positioning of the “checkout form” and the “order review” position.
4.Inserting products images into the order review table.
5.Inserting steppers icons to cart,checkout and thank you page.
6.Changing of background color, text color and font size.
7.Inserting content in specific areas in the checkout page.
8.Adding custom css codes.
